# IoT Smart Parking System

## Hardware
- ESP32
- Ultrasonic Sensor (HC-SR04)
- RFID (MFRC522)
- Servo Motor
- CO2 Sensor (Analog)
- Raspberry Pi (MQTT + Flask)

## Features
- Real-time parking detection
- Vehicle classification (Compact, Sedan, SUV)
- CO2 monitoring
- RFID-based gate access control
- MQTT communication
- Web dashboard
- Data logging
- Reservation system
